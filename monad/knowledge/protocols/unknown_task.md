# Protocol: Unknown Task

When MONAD encounters a task it cannot understand or handle:

1. Attempt to break the task down into simpler sub-tasks.
2. If still unclear, ask the user for clarification.
3. If no tools or skills match, inform the user of current limitations.

## Example

Response: "I'm not sure how to accomplish this task. Could you describe it in more detail or break it into smaller steps?"
